# Contenu à produire — CyberFit

## Défis (60 minimum pour rotation)

### Mots de passe (15)
- Créer un mot de passe fort
- Installer un gestionnaire de MDP
- Changer un MDP réutilisé
- Activer la génération automatique
- Vérifier la force de ses MDP actuels
- Supprimer les MDP enregistrés dans le navigateur
- Activer le remplissage auto sécurisé
- Créer une phrase de passe
- Auditer ses MDP avec le gestionnaire
- Configurer le verrouillage auto du gestionnaire
- Partager un MDP de manière sécurisée
- Activer les alertes de compromission
- Migrer vers un gestionnaire de MDP
- Séparer MDP perso et pro
- Configurer la récupération de compte

### 2FA (10)
- Activer 2FA Gmail
- Activer 2FA banque en ligne
- Sauvegarder les codes de backup
- Installer une app authenticator
- Activer 2FA sur les réseaux sociaux
- Remplacer SMS par app authenticator
- Activer 2FA sur son compte Apple/Google
- Configurer 2FA sur son gestionnaire de MDP
- Tester la récupération 2FA
- Activer 2FA sur les services de stockage cloud

### Réseaux sociaux (10)
- Réviser la confidentialité Facebook
- Limiter l'audience des publications Instagram
- Désactiver la localisation
- Auditer les apps connectées
- Vérifier les paramètres de tag
- Supprimer les anciennes publications publiques
- Restreindre qui peut vous trouver
- Désactiver le tracking publicitaire
- Vérifier les sessions actives
- Configurer les alertes de connexion

### Emails (10)
- Reconnaître un email de phishing
- Activer les filtres anti-spam
- Nettoyer les anciennes sessions
- Vérifier les règles de transfert (forwarding)
- Créer un alias email
- Désabonnement des newsletters non lues
- Configurer l'authentification renforcée
- Identifier les pièces jointes dangereuses
- Vérifier les appareils connectés à la boîte mail
- Activer le chiffrement des emails

### Appareils (10)
- Activer les mises à jour automatiques
- Chiffrer le disque dur
- Configurer un antivirus
- Auditer les permissions des apps
- Activer le verrouillage automatique
- Configurer la localisation à distance
- Désinstaller les apps inutilisées
- Vérifier les connexions Bluetooth/WiFi
- Sauvegarder ses données
- Configurer un DNS sécurisé

### Navigation (5)
- Installer un bloqueur de pubs
- Vérifier les extensions du navigateur
- Nettoyer les cookies
- Activer le mode HTTPS-only
- Configurer un moteur de recherche respectueux de la vie privée

## Guides (40 minimum)

### Répartition par niveau
- **Débutant** : 20 guides
- **Intermédiaire** : 15 guides
- **Avancé** : 5 guides

### Plateformes couvertes
- iOS
- Android
- Windows
- Mac
- Web (multi-navigateurs)

### Exemples de guides
- Activer 2FA sur Gmail (débutant, Web)
- Créer un mot de passe fort (débutant, tous)
- Repérer un email de phishing (débutant, tous)
- Configurer un gestionnaire de mots de passe (débutant, tous)
- Chiffrer son iPhone (intermédiaire, iOS)
- Configurer un VPN (intermédiaire, tous)
- Analyser les headers d'un email suspect (avancé, Web)
- Configurer un DNS-over-HTTPS (avancé, tous)
